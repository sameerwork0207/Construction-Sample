========================================
HOW TO ADD IMAGES:
========================================

1. Create a folder named "images" in the project root (if not already done)
   
2. Prepare your images:
   - Recommended format: JPG (jpg)
   - Resize to recommended dimensions
   - Optimize for web (compress if large)

3. Save images with exact filenames:
   - hero-kitchen.jpg
   - project1.jpg
   - project2.jpg
   - project3.jpg
   - project4.jpg

4. Place all images in the "images/" folder

5. Refresh your browser to see the images on the website

========================================
IMAGE QUALITY TIPS:
========================================

- Use high-quality images (at least 72 DPI)
- Ensure good lighting and clear visibility
- Use professional photography or renderings
- Maintain consistent style across all project images
- Avoid images with watermarks or logos
- Consider aspect ratios for better layout
